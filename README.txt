Here are the setup steps for the URLAutomatio



1. check if python us installed using "python --version"
	if not, then install it
	
2. Install the following python packages :
		selenium
		python-docx
		pillow
		python-dotenv
		pyautogui

pip install selenium python-docx pillow python-dotenv
pip install pyautogui

3. Run the 'URL_Automation' python script.	
	The script will create a directory 'TEMP_screenshots', where it will temporarily store all the screenshots
	The script will also create a docx file and add all of the screenshots in that file
